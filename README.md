# 📝 Blog App

A full-stack Blog Application that allows users to create, read, update, and delete blog posts. This project demonstrates modern web development practices including RESTful APIs, authentication, and responsive UI design.

---

## 🚀 Live Demo

👉 [https://github.com/LearnCodingOfficial/blog-app](https://github.com/LearnCodingOfficial/blog-app)

---

## 📌 Features

* ✍️ Create, edit, and delete blog posts
* 📖 View all blogs and individual posts
* 🔐 User authentication (Login/Signup)
* ❤️ Like and interact with posts
* 📱 Responsive design for all devices
* ⚡ Fast and optimized performance

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/LearnCodingOfficial/blog-app.git
```

### 2. Navigate to project folder

```
cd blog-app
```

### 3. Install dependencies

```
npm install
```

### 4. Setup environment variables

Create a `.env` file and add:

```
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 5. Run the application

```
npm start
```

## 📡 API Endpoints

| Method | Endpoint   | Description     |
| ------ | ---------- | --------------- |
| GET    | /posts     | Get all posts   |
| GET    | /posts/:id | Get single post |
| POST   | /posts     | Create new post |
| PUT    | /posts/:id | Update post     |
| DELETE | /posts/:id | Delete post     |
